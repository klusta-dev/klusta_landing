# Klusta Landing Page

Landing page for **Klusta** — find and list short-term rentals (shortlets) across Nigeria. Built with Next.js 14 and Tailwind CSS.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Structure

- `src/app/` — App Router: `layout.tsx`, `page.tsx`, `globals.css`
- `src/components/` — Section components: Header, Hero, DiscoverSection, EaseSection, WhyKlusta, HowItWorks, Footer

## Design

- **Fonts:** Playfair Display (headlines), Inter (body)
- **Colors:** Olive green (`#6D8E53`), beige (`#E8E2D7`), white, dark text
- **Content:** Adapted for Klusta (shortlets, Nigeria, list property, negotiate, book)
